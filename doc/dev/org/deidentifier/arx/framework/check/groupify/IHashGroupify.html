      • analyze

        void analyze(boolean force)
        Computes the anonymity properties and suppressed tuples etc. Must be called when all tuples have been passed to the operator. When the flag is set to true the method will make sure that all equivalence classes that do not fulfill all privacy criteria are marked as being suppressed. If the flag is set to false, the operator may perform an early abort, which may lead to inconsistent classification of equivalence classes.
        Parameters:
        force -
