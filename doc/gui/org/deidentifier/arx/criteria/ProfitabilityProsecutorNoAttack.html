    public class ProfitabilityProsecutorNoAttack
extends ImplicitPrivacyCriterion
    Privacy model for the "no-attack" variant of the game theoretic approach proposed in: A Game Theoretic Framework for Analyzing Re-Identification Risk. Zhiyu Wan, Yevgeniy Vorobeychik, Weiyi Xia, Ellen Wright Clayton, Murat Kantarcioglu, Ranjit Ganta, Raymond Heatherly, Bradley A. Malin PLOS|ONE. 2015.
