    public class ImportWizardPageColumns
extends org.eclipse.jface.wizard.WizardPage
    Column overview page This pages gives the user an overview of the detected columns and allows him to change things around. First of all columns can be enabled or disabled on an individual basis. Secondly the order of the columns can be changed around. Furthermore a data type along with a format string can be defined for each column. A single column is represented by ImportWizardModelColumn, the list of all detected columns can be accessed by ImportWizardModel.getWizardColumns().
