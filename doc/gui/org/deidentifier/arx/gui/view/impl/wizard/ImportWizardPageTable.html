    public class ImportWizardPageTable
extends org.eclipse.jface.wizard.WizardPage
    Table overview page This pages gives the user an overview of the detected tables and allows him to select the desired one by clicking on it. The tables itself are retrieved from ImportData#getJdbcTables()(). The selected one will be assigned via ImportWizardModel.setSelectedJdbcTable(String) along with the detected columns for this table using ImportWizardModel.setWizardColumns(List) and its preview data ImportWizardModel.setPreviewData(List).
