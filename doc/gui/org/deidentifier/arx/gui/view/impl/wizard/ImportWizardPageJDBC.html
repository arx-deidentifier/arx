    public class ImportWizardPageJDBC
extends org.eclipse.jface.wizard.WizardPage
    JDBC page This page offers means to specify connection details for a database. For now MySQL, PostgreSQL and SQLite is supported. In case of remote database types (i.e. MySQL and PostgreSQL) the user is asked for the server and a username and password. In case of SQLite the user can select any *.db file. After ther user specified the details a connection is established and passed on to ImportWizardModel. This includes:
