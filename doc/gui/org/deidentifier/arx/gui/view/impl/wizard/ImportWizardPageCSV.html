    public class ImportWizardPageCSV
extends org.eclipse.jface.wizard.WizardPage
    CSV page This page offers means to import data from a CSV file. It contains mechanisms to select such a file, and offers the user the ability to define the separator and whether or not the first row contains a header describing each column. A live preview makes sure the user will immediately see whether or not his choices make any sense. All of the data gathered on this page is stored within ImportWizardModel. This includes:
    • ImportWizardModel.setWizardColumns(List)
    • ImportWizardModel#setFirstRowContainsHeader(boolean)
    • {@link ImportWizardModel#setFileLocation(String)}
    • {@link ImportWizardModel#setCsvSeparator(char)}
