    public class MetricMDNUEntropyPrecomputed
extends AbstractMetricMultiDimensional
    This class provides an efficient implementation of the non-uniform entropy metric. It avoids a cell-by-cell process by utilizing a three-dimensional array that maps identifiers to their frequency for all quasi-identifiers and generalization levels. It further reduces the overhead induced by subsequent calls by caching the results for previous columns and generalization levels. TODO: Add reference
